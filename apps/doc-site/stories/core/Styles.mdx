# Styles

The styles property is present across many IoT App Kit visualizations that utilize queries to provided data to the widget.

Use this property to customize certain settings such as the display name, color from your data streams within your widget.



### Example

To display "Windmill 2000" in a bar chart:
```
<Chart
  queries={[
    query.timeSeriesData({
      assets: [{
        assetId: 'windmill-asset-2000',
        properties: [{ propertyId: 'wattage' }]
      }]
    })
 ]}
/>
```

Let's add to this example. To display the wattage output in orange and not the default blue hue:

1. Add a `refId` with the name 'wattage-styles'.
2. Add the `styles` property, which maps the name to the styles we want to associate with.

```
<Chart
  queries={[
    query.timeSeriesData({
      assets: [{
        assetId: 'windmill-asset-2000',
        properties: [{ propertyId: 'wattage', refId: 'wattage-styles' }]
      }]
    })
 ]}

  styles={{ 'wattage-styles': { color: 'orange' } }}
/>
```

This displays the wattage output, in orange text.
